# 503IT-Group-E-Group-__-Cyber-Smart-Game.-Cyber-shield-spot-the-phishing-scam
503IT Communication and Collaboration group project evidence for Cyber-Smart Gaming for Education.
Cyber Shield: Spot the Phishing Scam is an educational cyber safety game created by Group E by group cyber shield spot the phising game group. The aim of this project is to design an interactive game that teaches students how to stay safe online by identifying phishing scams, fake emails, and suspicious links. The game is designed for students aged 10–12 years old and focuses on improving online scam awareness and safe internet decision-making skills.

The game allows players to read realistic email scenarios, identify warning signs of phishing attempts, and choose the safest action from different options. Players receive instant feedback, earn points, and learn useful cyber safety tips throughout the game. The project includes features such as interactive quizzes, score tracking, badges, and kid-friendly visuals to make learning engaging and easy to understand.

Current progress for the project includes forming the group and assigning team roles on Day 1. On Day 2, the team completed the GitHub setup, finalized the cyber safety topic, and created the game canvas and storyboard for the project.
